== Installation ==  

It's very easy and straight forward to install this plugin. But before installing **Unlimited Elementor Inner Sections** make sure you've installed and activated [Elementor](https://wordpress.org/plugins/elementor/ "Install Elementor").

= Automatic Installation =
1.  Go to `Plugins > Add New` screen from your WordPress website dashboard.
2.  Search for `Unlimited Elementor Inner Sections By TaspriStudio`.
3.  Install and activate the plugin, that's it!

= Manual Installation =
1.  Download [Unlimited Elementor Inner Sections](https://downloads.wordpress.org/plugin/unlimited-elementor-inner-sections-by-taspristudio.zip "Unlimited Elementor Inner Sections").
2.  Extract the `unlimited-elementor-inner-sections-by-taspristudio.zip` file. You'll get plugin files inside `unlimited-elementor-inner-sections-by-taspristudio` directory.
3.  Upload the plugin files to the `/wp-content/plugins/unlimited-elementor-inner-sections-by-taspristudio` directory.
4.  Activate the plugin through the 'Plugins' screen form your WordPress website dashboard.

= Usage =
After activating **Unlimited Elementor Inner Sections** you're just 2 clicks away to add nesting sections/columns.

1. Open a page to edit in Elementor mode.
2. Right-click on a column where you want to have a nested section/column.
3. Click the `Add Nested Section` from the options menu.

That's it! You should have a nested column right away. You can manage these columns like any other Elementor column. Add a new column, use column settings and anything Elementor offers with a column.
